# InfraSense - IR Based Object Detection System

## Overview

InfraSense is a custom-built IR object detection system created by understanding the internal working of IR sensor modules.

The circuit detects nearby objects using infrared reflection and comparator-based signal processing.

## Working Principle

The IR LED emits infrared light continuously.

When an object comes near the sensor, the reflected IR signal is received by the photodiode.

The LM358 comparator compares the received signal voltage with a reference voltage and generates the detection output.
